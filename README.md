# penjelasan-Storm-chaser-versi-indo
Storm chaser adalah orang yang mengejar badai untuk mengamati fenomena cuaca ekstrem seperti tornado dan badai petir. Mereka menggunakan pengetahuan cuaca serta alat seperti radar dan kamera untuk mengikuti pergerakan badai secara langsung. Meski berisiko, kegiatan ini membantu penelitian dan meningkatkan kesadaran tentang bahaya alam sekitar kita.
